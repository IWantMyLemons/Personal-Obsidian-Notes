Definition:: A collection of data elements placed next to eachother

```c
int a[5] = {1,2,3,4,5};
// memory should look like
// value   [...][...][1  ][2  ][3  ][4  ][5  ][...][...]
// address [...][...][a  ][a+1][a+2][a+3][a+4][...][...]
```
